# Amaterest Site

Mini-site statique pour remplir les champs Pinterest Developer.

## Fichiers

- `index.html` : page d'accueil publique
- `privacy.html` : politique de confidentialité
- `icon.svg` : icône vectorielle
- `icon-512.png` : icône PNG 512 x 512 générée localement

## URLs à utiliser après déploiement

- Site Web de l'entreprise : `https://ton-domaine.com/`
- Politique de confidentialité : `https://ton-domaine.com/privacy.html`

Pinterest refuse généralement `localhost` dans le champ du site public. Le site doit être publié sur une URL `https://`.
