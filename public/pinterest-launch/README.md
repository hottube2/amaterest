# Pinterest Launch Assets

These files are the current official launch visuals used by Amaterest.

If a new revision is needed later, keep these exact filenames:

- `pin-01.jpg`
- `pin-02.jpg`
- `pin-03.jpg`
- `pin-04.jpg`
- `pin-05.jpg`
- `pin-06.jpg`
- `pin-07.jpg`
- `pin-08.jpg`
- `pin-09.jpg`
- `pin-10.jpg`

To replace them with a newer final revision:

1. keep the same filenames
2. overwrite the existing files
3. refresh `/pinterest-launch`

If one dedicated file is missing, Amaterest falls back to:

1. `PINTEREST_FALLBACK_IMAGE_URL`
2. `PINTEREST_TEST_IMAGE_URL`
3. `/icon-512.png`
